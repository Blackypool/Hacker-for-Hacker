# Hacker_for_Hacker

Программа запрашивает пароль, чтобы проверить его корректность нужно нажать клавишу энтер. При совпадении выводится надпись **success**, при неудаче **ha loh**. Паролем является комбинация клавиш, сумма скан-кодов которых = 100, например, ***PPPP***.

В коде присутсвует уязвимость, не связанная с коллизией проверки введенного пароля, позволяющая при некорректно введенном пароле вывести надпись **success**.

---

## Секрет 1 ошибки:
Все скан-коды нажатых клавиш сохраняются в отведенный буфер и лежат по четным ячейкам (по нечетным скан-коды отпускания клавиш). В конце буфера в ячейке 32-33 лежит канарейка, которая является комбинацией нажатой и той же отпущенной клавиши ***W***, а в самом конце лежит флаг, отвечающий за итоговый вывод. Таким образом, если правильно вычислить расположение птички и перезаписать её нажатием клавиши ***W***, получится перескочить через неё, так как при проверке она фактически не изменится. Затем, если нажать клавишу ***Esc*** запишется скан-код 01h, что в конечном итоге приведет к выводу на экран надписи **success**.
